## Como Rodar

### Pré-requisitos
- [Nodejs](https://nodejs.org/en/) (v14 ou superior)

### Rodando o projeto

Na raiz do projeto execute os seguintes comandos:

```bash
# Instalando dependências
$ npm install
# Rodando o projetos
$ npm start
# Rodando script de testes
$ npm test
```
### Requisições
```bash
# endpoints disponíveis: POST
  /rgbToHex
  /hextoRgb

# Exemplo de body JSON
{
    "rgb": "255,255,255",
    "hex": "#ffffff"
}
```
